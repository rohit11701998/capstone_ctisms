----------------------------------------------------------------
STEP 1 — INSTALL XAMPP
----------------------------------------------------------------
Download: https://www.apachefriends.org/download.html
Install and open XAMPP Control Panel.
START: Apache   and   MySQL

----------------------------------------------------------------
STEP 2 — COPY FILES
----------------------------------------------------------------
Extract this ZIP. Copy the ctisms_v1 folder to:
  Windows: C:\xampp\htdocs\ctisms_v1\
  macOS:   /Applications/XAMPP/htdocs/ctisms_v1/

----------------------------------------------------------------
STEP 3 — CREATE DATABASE
----------------------------------------------------------------
Open: http://localhost/phpmyadmin
1. Click "New"
2. Database name: ctisms_v1   Collation: utf8mb4_unicode_ci
3. Click Create
4. Select ctisms_v1, click Import tab
5. Choose File → select database.sql → click Import

----------------------------------------------------------------
STEP 4 — RUN SETUP
----------------------------------------------------------------
Open: http://localhost/ctisms_v1/setup.php
This creates the demo account with a working password.

----------------------------------------------------------------
STEP 5 — LOGIN
----------------------------------------------------------------
