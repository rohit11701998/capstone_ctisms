STEP 1 — INSTALL XAMPP
----------------------------------------------------------------
Download: https://www.apachefriends.org/download.html
Install, open XAMPP Control Panel.
START: Apache   and   MySQL

----------------------------------------------------------------
STEP 2 — COPY FILES
----------------------------------------------------------------
Extract this ZIP. Copy the ctisms_v2 folder to:
  Windows: C:\xampp\htdocs\ctisms_v2\
  macOS:   /Applications/XAMPP/htdocs/ctisms_v2/

----------------------------------------------------------------
STEP 3 — CREATE THE DATABASE
----------------------------------------------------------------
Open: http://localhost/phpmyadmin
1. Click "New"
2. Name: ctisms_v2   Collation: utf8mb4_unicode_ci
3. Click Create
4. Select ctisms_v2, click the Import tab
5. Choose File → select database.sql → click Import

----------------------------------------------------------------
STEP 4 — RUN SETUP (REQUIRED)
----------------------------------------------------------------
Open: http://localhost/ctisms_v2/setup.php
Wait for the green success screen.
This generates correct password hashes on your PHP installation.

----------------------------------------------------------------
STEP 5 — LOGIN AND TEST
----------------------------------------------------------------
Open: http://localhost/ctisms_v2/

Suggested test flow:
  1. Login as Customer → Submit a ticket
  2. Login as Admin → Assign ticket to Tech Alice
  3. Login as Tech Alice → View assigned ticket, update status,
     add a comment
  4. Login as Customer → See the reply + notification bell
